Excel Data Cleaning

Turned a messy, inconsistent dataset into an analysis-ready table by standardizing categories, fixing dates, removing duplicates, and polishing text with Excel functions.

Before

After

Objective

Clean, standardize, and structure a raw dataset so it is accurate, consistent, and ready for dashboards, reports, and analysis.

Key Cleaning Steps

  • Consolidated categories by spotting and correcting misspellings/variants (e.g., Democractic → Democratic), ensuring consistent groupings.
  • Fixed date column by converting mixed or text dates to a single, valid date type and format (e.g., using DATEVALUE, TEXT, and custom formats).
  • Removed duplicates using Excel’s Remove Duplicates to keep unique records.
  • Trimmed extra spaces with =TRIM(cell) to remove leading/trailing/double spaces that break lookups and grouping.
  • Standardized case with =PROPER(cell) for clean, readable names and categories.

Techniques & Functions Used

  • Text cleanup: TRIM, PROPER, SUBSTITUTE
  • Date normalization: DATEVALUE, TEXT, custom date formats
  • De-duplication: Remove Duplicates
  • Validation & control: Data Validation lists for categories
  • Quality checks: Filters, Conditional Formatting to spot remaining issues

Impact

  • Reliable, analysis-ready dataset free of spelling drift, text clutter, and duplicate rows.
  • Consistent dates for accurate time-series and grouping.
  • Faster downstream work—no rework or ad-hoc fixes needed during reporting.
